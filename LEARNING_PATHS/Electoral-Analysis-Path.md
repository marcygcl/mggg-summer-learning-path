# Electoral Analysis Learning Path

Structured path for learning electoral analysis replication methodology.

## Who This Is For

- Interns starting electoral analysis work
- People learning from census data to RCV/STV modeling
- Anyone replicating MGGG's methodology for new jurisdictions

## Time Investment

**Total:** ~7 weeks, ~40 hours

## Prerequisites

- Basic understanding of:
  - U.S. electoral system (plurality voting)
  - Census data existence
  - Version control (Git) - optional but helpful
- Comfort with:
  - Reading technical documentation
  - Learning new tools
  - Asking questions

## Learning Path Phases

### Phase 1: Foundations (Weeks 1-2)
**Goal:** Understand the building blocks

Topics:
- Census data fundamentals (VAP/CVAP)
- Demographic categories and their construction
- Electoral systems (plurality, IRV, STV)
- MGGG's general methodology

Resources:
- VAP-CVAP.pdf guide
- Lowell Report (Ranked Choice Voting analysis)
- Massachusetts RCV Report
- NotebookLM video summaries

### Phase 2: Data Collection (Week 3)
**Goal:** Find and organize data for your jurisdiction

Topics:
- Census data sources (IPUMS NHGIS)
- Electoral results databases
- Geographic data (shapefiles)
- Data validation and cleaning

### Phase 3: Methodology Definition (Week 4)
**Goal:** Understand and document your approach

Topics:
- Parameter definition (blocs, turnout, cohesion)
- Assumptions and limitations
- Simulation design
- Model selection (Plackett-Luce, Bradley-Terry, Cambridge)

### Phase 4: Implementation & Analysis (Weeks 5-6)
**Goal:** Run simulations and analyze results

Topics:
- GerryChain/VoteKit setup
- Running simulations
- Interpreting results
- Building visualizations

### Phase 5: Reporting (Week 7)
**Goal:** Communicate findings professionally

Topics:
- Report structure and flow
- Table and visualization standards
- Writing for non-experts
- Caveats and limitations

## Key Resources Needed

- MGGG Reports (Lowell, Massachusetts)
- GerryChain documentation
- VoteKit documentation
- Census Bureau resources
- Electoral data sources

## Success Criteria

You will know you've succeeded when you can:

1. ✅ Explain VAP/CVAP construction from Census data
2. ✅ Compare electoral systems and their representation effects
3. ✅ Find and validate data for a new jurisdiction
4. ✅ Define appropriate parameters for analysis
5. ✅ Run electoral simulations and interpret results
6. ✅ Write a professional report on your findings
7. ✅ Adapt MGGG's methodology to new contexts
