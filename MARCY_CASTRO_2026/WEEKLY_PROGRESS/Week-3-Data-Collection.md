# Week 3: Data Collection

**Status:** 🟡 In Progress  
**Focus:** Finding and organizing data for Kansas City analysis

## Overview

Week 3 is focused on finding and collecting the data needed for the Kansas City analysis. This includes census data (VAP/CVAP), electoral results, and geographic boundaries.

## Data Needs for KC

### Census Data
- [ ] VAP by race/ethnicity (2020 Census)
- [ ] CVAP estimates (ACS 5-year)
- [ ] Population by demographic categories (BVAP, HVAP, AVAP, WVAP)

### Electoral Data
- [ ] Historical election results (last 3-4 cycles if available)
- [ ] Candidate information and endorsements
- [ ] Precinct or tract-level results (if available)

### Geographic Data
- [ ] KC city boundaries (shapefile)
- [ ] Current council district boundaries
- [ ] Census tract/block group shapefiles
