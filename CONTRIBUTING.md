# Contributing Guide

How to contribute to and maintain this learning path repository.

## Quick Edits (GitHub Web)

1. Navigate to the file you want to edit
2. Click the pencil icon (✏️)
3. Make changes
4. Click "Commit changes"
5. Add a descriptive message

## Local Development

```bash
git clone https://github.com/marcygcl/mggg-summer-learning-path.git
cd mggg-summer-learning-path

# Make edits to .md files

git add .
git commit -m "Your descriptive message"
git push origin main
```

## File Structure Guidelines

- Use Markdown (.md) for all documentation
- Keep files organized by topic
- Include clear headings and sections
- Add links to related content
- Use Spanish with English technical terms

## Markdown Formatting

- `# H1` for main titles
- `## H2` for sections
- `### H3` for subsections
- `- bullet points`
- `1. numbered lists`
- `**bold**` for emphasis
- `[link text](url)` for links

## Adding New Content

1. Create file in appropriate folder
2. Follow existing formatting style
3. Link from related pages
4. Update CHANGELOG.md
5. Commit and push
