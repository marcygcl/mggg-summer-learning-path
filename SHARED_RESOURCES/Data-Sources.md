# Data Sources for Electoral Analysis

Where to find census, electoral, and geographic data.

## Census Data

### IPUMS NHGIS

**URL:** https://www.nhgis.org/  
**Type:** Historical U.S. Census data  
**Coverage:**
- 2020, 2010, 2000, 1990 Censuses
- Various earlier censuses
- Pre-1990: incomplete (Metropolitan areas only)

**What it provides:**
- VAP by Census tract
- CVAP (with estimation)
- Race/ethnicity breakdowns (P3, P4 tables)
- Population counts

**Access:** Free with registration  
**Cost:** Free

### American Community Survey (ACS)

**URL:** https://www.census.gov/acs  
**Type:** Ongoing Census Bureau surveys  
**Coverage:** Annual estimates, 5-year rolling averages

**What it provides:**
- Citizenship rates (for CVAP calculation)
- Detailed demographic data
- Economic and social characteristics

**Access:** Free via Census Bureau  
**Cost:** Free

## Electoral Results

### OpenElections

**URL:** https://openelections.net/  
**Type:** Precinct-level election results  
**Coverage:** Varies by state; improving

**What it provides:**
- Precinct-level results
- Federal, state, local elections
- Historical data

**Access:** Free, open source  
**Cost:** Free

### mggg-states Repository

**URL:** https://github.com/mggg-states  
**Type:** Precinct shapefiles + election data  
**Coverage:** Growing

**What it provides:**
- Cleaned election data
- Precinct boundaries (shapefiles)
- Compiled by MGGG

**Access:** Free, GitHub  
**Cost:** Free

### State Election Boards

**Type:** Official state-level election data  
**Coverage:** Varies by state

**What it provides:**
- Official election results
- Candidate information
- Often precinct-level

**Access:** Varies; contact directly  
**Cost:** Usually free

## Geographic Data

### U.S. Census Bureau TIGER/Line

**URL:** https://www.census.gov/geographies/mapping-files/time-series/geo/tiger-line-file.html  
**Type:** Shapefiles for Census geographies

**What it provides:**
- Census tracts
- Block groups
- Blocks
- Current and historical

**Access:** Free download  
**Cost:** Free

### OpenStreetMap

**URL:** https://www.openstreetmap.org/  
**Type:** Community-maintained geographic data

**What it provides:**
- City boundaries
- Precinct boundaries (sometimes)
- Street data

**Access:** Free  
**Cost:** Free
