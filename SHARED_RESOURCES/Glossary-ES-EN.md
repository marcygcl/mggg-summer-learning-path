# Glossary: English-Spanish Electoral Analysis Terms

| English | Español | Definition |
|---------|---------|-----------|
| VAP | Población en Edad de Votar | Population 18+ (citizens + non-citizens) |
| CVAP | Población Ciudadana en Edad de Votar | Population 18+ AND citizens only |
| BVAP | Población Negra en Edad de Votar | Black Voting Age Population |
| HVAP | Población Hispana en Edad de Votar | Hispanic Voting Age Population |
| AVAP | Población Asiática en Edad de Votar | Asian Voting Age Population |
| WVAP | Población Blanca en Edad de Votar | White Voting Age Population |
| Plurality | Pluralidad / Votación Plural | Winner-take-all system; most votes wins |
| IRV | Voto por Preferencia Instantánea | Instant Runoff Voting; ranked choice single-winner |
| STV | Voto Transferible Único | Single Transferable Vote; ranked choice multi-winner |
| Ballot | Papeleta / Voto | One person's vote cast in an election |
| Preference Profile | Perfil de Preferencias | Distribution of voter rankings across candidates |
| Cohesion | Cohesión | Degree to which a group votes together for same candidate |
| Droop Quota | Cuota Droop | Threshold for seat allocation in STV systems |
| Precinct | Precinto Electoral | Smallest electoral geographic unit for voting |
| Census Tract | Censo Tract | Census geographic unit (~1,200-8,000 people) |
| Shapefile | Shapefile | Geographic data file format for boundaries |
| IPUMS NHGIS | IPUMS NHGIS | Database of historical U.S. Census data |
| ACS | American Community Survey | Ongoing Census Bureau demographic survey |
| Ecological Inference | Inferencia Ecológica | Statistical method inferring group voting from aggregate data |
| Candidate Strength | Fortaleza del Candidato | Relative electoral appeal/viability of a candidate |
